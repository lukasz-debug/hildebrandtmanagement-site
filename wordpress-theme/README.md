# Motyw WordPress: Hildebrandt Management

Ten katalog zawiera gotowy motyw WordPress odpowiadający stronie firmowej.

## Instalacja
1. Spakuj katalog `hildebrandt-management` do pliku ZIP.
2. W panelu WordPress przejdź do **Wygląd → Motywy → Dodaj nowy → Wyślij motyw**.
3. Wgraj ZIP i aktywuj motyw.
4. Ustaw stronę główną jako statyczną (Ustawienia → Czytanie), jeśli chcesz wymusić `front-page.php`.

## Struktura
- `hildebrandt-management/style.css` — metadane motywu i style.
- `hildebrandt-management/functions.php` — konfiguracja motywu i enqueue stylów.
- `hildebrandt-management/front-page.php` — główny landing page.
- `hildebrandt-management/index.php` — fallback listy wpisów.
- `hildebrandt-management/header.php` / `footer.php` — wspólna struktura HTML.
